# Premier League Player Stats Explorer

## [Website Link](https://siddharth114-playerstats-player-stats-bqztwc.streamlit.app/)

---

### This app performs webscraping of Premier League player stats. The statistics of players are taken from Football-reference.com.

### A glossary has been added to help users identify what the statistics mean.
